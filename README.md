# jsproject1-
Click-to-Draw Canvas with Undo/Redo/Reset
A browser-based drawing tool built entirely with vanilla JavaScript and HTML5 Canvas — no frameworks, no backend, no dependencies. Users click anywhere on the canvas to place a randomly coloured dot at that position. The app maintains a full history of canvas states, letting users undo or redo any action, or reset the canvas entirely — via on-screen buttons or keyboard shortcuts (Ctrl+Z, Ctrl+Y, Ctrl+R).
Key features:

Random colour generation on every click
State history management using Canvas ImageData snapshots
Undo/redo stack with a capped history limit (memory-safe)
Keyboard shortcuts alongside clickable buttons
Touch support for mobile devices
Single HTML file — runs directly in any browser, no server or build step needed

Tech used: HTML5, CSS3, JavaScript (Canvas API)
